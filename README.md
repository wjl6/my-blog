# 我的技术博客

欢迎来到我的技术博客仓库！这里记录了我的面试经验、学习历程、参与开源项目的经验以及自己开源项目的开发心得。

## 📚 关于我

这里是我记录技术成长的地方，包括：
- 📈 **面试题目**：聚焦技术求职与能力沉淀，系统整理高频面试真题
- 📖 **学习笔记**：记录在学习新技术、新框架过程中的心得体会
- 🤝 **开源贡献**：分享参与开源项目的经历和收获
- 🚀 **项目实践**：记录自己开源项目的开发过程、遇到的问题和解决方案

## 📁 仓库结构

```
.
├── README.md                       # 本文件
```    

## 📝 文章分类

### 面试题目 (interview/)
聚焦 真实面试场景，系统整理技术岗位高频面试题，内容来源于：
- 个人真实面试经历
- 高频面试考点总结
- 实际项目中的技术延展

### 学习笔记 (learning/)
记录在学习新技术、框架、工具等过程中的笔记和总结。

### 开源贡献 (open-source-contributions/)
分享参与开源项目的经历，包括：
- 如何开始参与开源项目
- 贡献过程中的心得体会
- 遇到的挑战和解决方案
- 与开源社区互动的经验

### 我的项目 (my-projects/)
记录自己开源项目的开发历程：
- 项目构思和设计思路
- 开发过程中的技术选型
- 遇到的问题和解决方案
- 项目迭代和优化经验

## 🚀 如何使用

1. **阅读文章**：直接浏览 `posts/` 目录下的 Markdown 文件
2. **查找内容**：使用 GitHub 的搜索功能或按分类目录查找
3. **反馈交流**：欢迎通过 Issue 或 Pull Request 进行交流
